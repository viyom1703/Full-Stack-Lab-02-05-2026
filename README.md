# 🔗 URL Shortener

A simple backend project that converts long URLs into short links and redirects users using Node.js, Express, and MongoDB.

---

## 🚀 Features

- Generate short URLs
- Redirect to original URLs
- Store data in MongoDB
- Simple REST API

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- shortid
